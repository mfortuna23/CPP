# CPP Modules

## üìñ Overview

CPP Modules is a collection of progressive C++ projects developed as part of the 42 School curriculum. The goal of this project is to introduce and deepen understanding of modern C++ (C++98) concepts through hands-on exercises, moving from basic syntax to advanced object-oriented and generic programming principles.

Each module focuses on a specific set of C++ features and best practices, reinforcing strong design, memory safety, and code readability.

| Module    | Topics Covered                                                     |
| --------- | ------------------------------------------------------------------ |
| **CPP00** | Namespaces, classes, member functions, std::string, I/O streams    |
| **CPP01** | Memory allocation, references, pointers, switch statements         |
| **CPP02** | Ad-hoc polymorphism, operator overloading, Orthodox Canonical Form |
| **CPP03** | Inheritance, access specifiers, virtual functions                  |
| **CPP04** | Subtype polymorphism, abstract classes, interfaces                 |
| **CPP05** | Exceptions, error handling, nested classes                         |
| **CPP06** | Type casting (`static_cast`, `dynamic_cast`, etc.)                 |
| **CPP07** | Templates, generic programming                                     |
| **CPP08** | STL containers, iterators                                          |
| **CPP09** | Advanced STL usage, parsing, algorithms, performance optimization  |
